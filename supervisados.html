Supervised learning

The supervised algorithms are based on the existence of one or more input variables (X), a variable of output (Y). An algorithm is used to learn the function it maps:

The objective is to approximate the mapping function so well that when new information is available, the model can be well estimated.
They are named supervised because the model "learns", the correct answers are known since the data is labeled, therefore the algorithm iteratively makes predictions and is corrected with the outputs that are known to be correct.

Types of supervised algorithms

The supervised algorithms can be both regression, if the output obtained by the model is numeric as classification if it is a class:

  • Regression
  • Classification
