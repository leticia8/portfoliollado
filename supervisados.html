Supervisados

Los algoritmos supervisados se basan en la existencia de una o más variables de entrada (X), una variable de salida (Y) y se utiliza un algoritmo para aprender la función que mapea El objetivo es aproximar tan bien la fufnción de mapeo que cuando se cuente con nueva información se pueda estimar bien el modelo. Se nombran supervisados porque el modelo "aprende", se conocen las respuestas correctas, el algoritmo hace predicciones iterativamente y es corregido con las salidas que se saben correctas.

Tipos de algoritmos supervisados

  • De regresión
  • De clasificación
