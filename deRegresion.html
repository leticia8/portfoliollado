Regression algorithms

Regression problems are those in which the output variable takes a continuous value.
Most used regression algorithms are:

LINEAR REGRESSION

It is a linear model, which assumes a linear relationship between the input variables and the output variable .
The linear regression equation assigns a scale factor to each input attribute, called a coefficient and usually represented by the letter Î². An additional coefficient is added, giving a greater degree of freedom usually called the intercept or the bias coefficient. It is common to talk about the complexity of the linear regression model referring to the number of coefficients.
Learning a linear regression model applies estimating the values of the coefficients used in the representation of the information.

DATA PREPARATION:
Assumption of linear relationship: it may be necessary to transform data to make the relationship between input and output linear.
Noise and collinearity must be removed
Gaussian distribution: predictions will be more reliable if the distribution is Gaussian (normal).
Normalization: predictions will be more reliable if normalization is used.

DECISION TREES - CART - Classification And Regression Trees

They have been classically referred to as decision trees although currently they are called Classification and Regression Trees (CART)
The representation is a binary tree where each node represents an input variable and a cut-off point in said variable. The leaves of the tree contain the output variable that are used to make the prediction.
Making predictions is pretty straightforward. Each input variable can be thought of as a dimension in p-dimensional space. The decision tree divides the space into two rectangles or hyper-rectangles with more entries.
An avid method is used to divide the space called binary recursive division. It is a numerical procedure where all values are aligned and different division points are tested using a cost function. The division with the best cost is selected.
For regression problems the cost function that is minimized to select the cut-off points is the sum of the squared errors of all the training samples.
The most common stopping procedure is to use the minimum amount of training data used for each leaf node.
The tree can be pruned after training since simpler trees are preferred since they are easier to understand and less likely to overfit information.
The quickest and simplest pruning methods are carried out by evaluating the effect on the cost of removing said node from the model. When improvements are not achieved, the procedure is stopped.

DATA PREPARATION:
No preparation required.

RANDOM FOREST

They are an improvement over decision trees that use the Bagged (Bagging and Aggregation) algorithm. Is an avid algorithm, this implies that in each iteration it looks for the local optimum that may end up not being the optimum at a general level. In each split, a number of attributes established by parameter are evaluated that are randomly selected from among all the available attributes, It is recommended for regression algorithms that this number be equal to p / 3, where p is the number of variables or input attributes.
Random Forest solves the problem of correlation between models by including randomness in their elaboration. To decide the number m of attributes to select, it is better to start with 5 values between 2 and p (number of attributes or input variables) and go testing the performance, for the number of models to run. An approximate 1000 trees is recommended, although as in all cases it depends on the problem and knowledge of the business. for In random forest all trees are created independently to have the desired depth and contribute in the same way to the model.

GRADIENT BOOST FOR REGRESSION TREES

Given a loss measurement function and a weak learning algorithm, an additive model is found that minimizes the loss function.
It starts with the average of the responses, the gradient is calculated and the model is adapted to the residuals to minimize the loss function. The model is added to the previous one and the procedure continues as many times as stipulated in the parameter that is established by the researcher.
Advantages of using trees is that they can become weak models by controlling the depth of the tree, in turn, they can be separated, maintaining independence and can be generated quickly.
